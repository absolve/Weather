# Weather
一个简单的天气app
这个app是2016年的时候开发的之后很久没有改过，最近便找到放在github上面，更新了项目的配置。  
使用openweather天气，和风天气预报，高德地图，一些key需要自己配置。  
之前一些代码比较久了，需要慢慢的更改。  

最近修改了原来的界面,新版的天气app使用了openweather天气，实现自动定位获取天气信息，如果需要运行需要自己配置地图的key和openweather天气key
